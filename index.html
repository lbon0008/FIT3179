Electric Vehicles Around the World 2010-2020

By Luke Bonso

Number Of Charing Ports Around The World

The Map displays the number of EV charging ports Per
country. As the number of EVs a country has grows,
so too should its capacity to charge these vehicles.

Hover over any of the countries which are filled
in with colour to see specific information about said
country.

EV Sales from 2010 to 2020

This depecits how the sales of EVs has changed over
time for each differnt country. This plot can be filtered
by clicking on the legend to see the line plot
for a specific country. It can be seen that the
sales of EVs has dramatically increased for all countries
within this time period.

EV Stock from 2011 to 2020

This plot shows how the stock of electric vehicles is
allocated to each country of the world. This plot is
filterable by year so it can be seen how the
stock level changes relative to each country.

EV Sales vs Charging Points from 2011 to 2020

This plot shows the relationship between the sales of EVs
and the number of charging Points avaible for use for
each country. This plot is filterable by year so that
it can be seen how this relationship changes over time.
This plot indicates the change in infrastructure to deal with
more EVs on the road.

BEV vs PHEV from 2010 to 2020

This plot depicts how the stocks of Battery Electric Vehicles
compares to the stocks of Plug in Hybrids has changed
over time for the various countries. This gives insight into
what types of EVs have been most popular for each
country in each year.
